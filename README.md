GrainStorageDemo

A client-side web application for managing grain storage facilities. It allows operators to organize storage compartments, track grain types, batch records, fill status, and temperature per partition — all without a backend in this Demo.

Tech Stack: Vanilla JavaScript (ES6 modules), HTML5, CSS3, localStorage

Key Features:

Login with demo credentials
Create and manage storage units with configurable grid layouts
Per-partition tracking: grain type, batch info, fill status (empty / full / cleaned / reserved), and temperature
Separate hose/tube management view
Weight and batch record logging
Archive import functionality
Mobile-responsive UI
